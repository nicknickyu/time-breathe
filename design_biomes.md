# 地形地貌设计方案

## 核心原则
- 每局仍为 3 种地形，不增加单局复杂度
- 不同局的 3 种地形组合不同，构建出完全不同的地貌
- 每种组合有自己独特的演化规则、动物和视觉风格

## 方案A：水 + 草 + 山（现有 → 森林湿地）
水源催生草地，山地拔高阻挡水流，形成高低错落、溪流穿插的自然地貌。
- 动物：野牛（草地）、猫头鹰（高山）、河马（水）

## 方案B：沙 + 仙人掌 + 绿洲（沙漠绿洲）
- 沙地会扩张但收益低
- 绿洲缓慢扩水
- 仙人掌需要聚集才能生长
- 玩家要控制沙化、规划绿洲位置
- 动物：骆驼（沙地）、蜥蜴（岩石）、鹭（绿洲水边）

## 方案C：雪 + 冻土 + 温泉（冰原）
- 冻土限制扩张速度
- 温泉产生热气融化周边冻土
- 雪覆盖其他地形
- 玩家围绕温泉展开，冰与火的对抗感
- 动物：企鹅（雪地）、驯鹿（冻土）、鱼（温泉）

## 方案D：灰烬 + 熔岩 + 水晶（火山）
- 熔岩蔓延烧毁一切
- 灰烬冷却后变为可用的地
- 水晶只能在远离熔岩的安全处生长
- 争夺生存空间的紧张感
- 动物：火蜥蜴（熔岩边）、岩羊（高地）、晶蝶（水晶附近）

## 方案E：蘑菇 + 腐木 + 苔藓（菌林）
- 腐木缓慢释放孢子→生成蘑菇
- 苔藓覆盖腐木扩大范围
- 潮湿昏暗的密林感
- 动物：萤火虫（蘑菇群）、蜗牛（苔藓）、鼯鼠（腐木）

## 后续规划
- 可做成**地貌图鉴收集系统**
- 每次开局随机抽到一种地貌
- 玩家通过不同地貌解锁图鉴收集
