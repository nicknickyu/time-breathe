# v3 污染对抗 + 关卡设计

## 核心玩法追加
玩家初始面对的不是空棋盘，而是被**污染的地形**。在演化过程中，玩家放置的地形与污染之间存在对抗：
- 污染会腐蚀玩家放的格子
- 玩家需要通过放置策略规避/净化污染
- 目标：让尽可能多的地形变成正常格子，吸引小动物入住

## 关卡设计维度

### 棋盘结构
通过预设障碍物（岩石、深水、废墟）改变棋盘有效区域，玩家无法在上面放置。

### 污染布局（核心变量）
- 污染源数量：1个集中源 / 3-4个分散源
- 污染形状：条状 / 点状 / 边缘向内侵蚀
- 污染等级：轻度（1格净化）、重度（需组合净化）
- 扩散方向：固定风向 / 均匀 / 沿低地势

### 可用地形组合
不同关卡限制可用地形种类，改变策略。

### 目标多样化
- 净化型：净化率达到80%即可过关
- 生物型：必须吸引到特定动物入住
- 经济型：在污染侵蚀超过50%前拿到足够分数
- 防守型：坚持N回合不让污染触碰到指定区域

### 污染行为模式
- 静态污染：不扩散，需主动净化 → 纯解谜
- 脉动污染（潮汐）：每N回合扩散一次 → 节奏感
- 智能污染：优先侵蚀玩家刚放的地形 → 心理博弈
- 潮汐污染：全屏污染周期性涨退 → 窗口期操作

## 关卡示例

### 关卡1 "新手"
小棋盘，单污染源，全地形可用，目标：净化100%

### 关卡2 "潮汐" ✅（用户选中）
污染每3回合一涨潮，玩家要趁退潮时净化，目标：存活10回合

### 关卡3 "堡垒" ✅（用户选中）
棋盘中间有一块"净土"，污染从四面向中间侵蚀，目标：保住净土不沦陷
