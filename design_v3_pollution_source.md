# v3.2 污染源机制

## 核心改动
取消全棋盘均匀出现污染的方式，改为棋盘上生成 **1-3 个污染源**。

## 污染源特性
- 污染源是一个特殊格子，无法被地形覆盖
- 每个 tick 污染源向相邻格子扩散污染
- 玩家不能直接放置地形在污染源上
- 生成时自动避开棋盘边缘
- 污染源头顶显示当前**包围进度数字**（0/6）

## 清除污染源的条件
- **污染源相邻的 6 格全部被地形占据**（包围）时，污染源被清除
- 清除后，污染源消失，该格变为空地
- 清除后不再产生新污染

## 清除后的剩余污染
- 已扩散的污染仍保留在棋盘上需要净化
- 但污染不会继续扩散，玩家只需逐步清理即可

## 玩家的目标链条
1. 观察棋盘，识别污染源位置
2. 规划防线，在污染扩散路径上放置地形
3. 逐步推进包围圈，蚕食污染区域
4. 围住污染源（6邻格全满）→ 清除污染源
5. 打扫战场：清理剩余的残留污染

## 与现有机制关系
- 地形演化规则不变
- 动物入住规则不变
- 仅替换污染生成方式

## 关卡参数
- 污染源数量：1 / 2 / 3 个
- 污染源强度：扩散速度 / 扩散范围

## v3.3 抽牌Cost系统

### 核心概念
- 每个关卡有一个固定的 **总 Cost** 预算
- 玩家抽取地形牌时需要消耗 Cost
- 不同地形价格不同
- 清除污染源可获得额外 Cost

### 地形定价参考
| 地形 | Cost | 理由 |
|------|------|------|
| 草地 | 1 | 最便宜，演化中可自然生成 |
| 森林 | 2 | 能催生草地，生态价值高 |
| 水源 | 3 | 催生草地+阻断污染 |
| 山地 | 4 | 抗污染最强+可拔高 |

### Cost 流向
```
关卡初始 → 抽取地形消耗 → 清理污染源奖励 →
→ 更多 Cost → 抽更强地形 → 更快清污染源
```

### 与抽牌系统的结合
- 原先生成 3 组随机地形供选择 → 改为生成若干组含 Cost 价格的卡牌
- 玩家不仅要考虑"这组地形好不好"，还要考虑"我剩下的 Cost 够不够"
- 低价地形（草地）虽然弱，但能让玩家多抽几次、铺得更快
